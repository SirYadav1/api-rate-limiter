# ⚡ API Rate Limiter Dashboard

Track and monitor your API usage in real-time. Never hit rate limits unexpectedly.

![Dark Mode](https://img.shields.io/badge/theme-dark-light)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- 🔑 **API Key Management** — Add, monitor, and manage multiple API keys
- 📊 **Real-time Charts** — Usage over time + requests by service
- ⚠️ **Smart Alerts** — Warning at 80%, blocked at 100%
- 🌙 **Dark/Light Mode** — Toggle with one click
- 📤 **Export/Import** — Backup your data as JSON
- 📱 **Responsive** — Works on mobile and desktop
- 💾 **Local Storage** — All data stays in your browser

## Quick Start

1. Open `index.html` in your browser
2. Add your API keys with rate limits
3. Simulate requests or track real usage
4. Monitor alerts and stay under limits

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Chart.js 4.x
- LocalStorage API

## Use Cases

- Track OpenAI/Claude API spending
- Monitor GitHub API rate limits
- Watch third-party API quotas
- Prevent unexpected billing
