# API Rate Limiter Dashboard

Monitor your API usage in real-time. Never hit rate limits unexpectedly.

## Features

- **26 API Presets** — OpenAI, Claude, Gemini, GitHub, Stripe, Discord, and more
- **Rate Limit Tracking** — Real-time usage with progress bars
- **Smart Alerts** — Warning at 80%, blocked at 100%
- **Cooldown Timer** — Auto-reset when limit expires
- **Reset Windows** — Per minute, per hour, or per day
- **Request Log** — Filterable log of all simulated requests
- **Dark/Light Mode** — Toggle with one click
- **Export/Import** — Backup data as JSON
- **Burst Simulation** — Test rate limits with no delay
- **Responsive** — Works on mobile and desktop

## Quick Start

1. Open `index.html` in your browser
2. Click any preset to add an API key
3. Simulate requests to see rate limiting in action
4. Monitor alerts and cooldowns
