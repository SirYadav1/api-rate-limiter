# Contributing

Contributions are welcome! Here's how you can help.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/api-rate-limiter.git`
3. Install dependencies: `npm install`
4. Start dev mode: `npm run dev`

## Development

- Backend: Edit `server.js` or files in `routes/`
- Frontend: Edit files in `public/`
- Database: Schema is in `db.js`

## Pull Requests

1. Create a feature branch: `git checkout -b feature/my-feature`
2. Make your changes
3. Test: `npm start` and verify in browser
4. Commit: `git commit -m "feat: add my feature"`
5. Push: `git push origin feature/my-feature`
6. Open a Pull Request

## Code Style

- Use camelCase for JavaScript variables
- Use kebab-case for CSS classes
- Keep functions small and focused
- Add comments for complex logic

## Reporting Issues

- Use GitHub Issues
- Include steps to reproduce
- Include browser/Node.js version
- Add screenshots if applicable
